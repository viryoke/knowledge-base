# Color Palettes for HTML Slides

Tested color schemes for professional presentations.

## Ocean Teal (Recommended Default)

```css
:root {
  --dark: #0f172a;
  --primary: #0d9488;
  --secondary: #14b8a6;
  --accent: #5eead4;
  --text: #334155;
  --text-light: #64748b;
  --bg: #f8fafc;
  --card: #ffffff;
}
```

## Usage Tips

1. Dark slides: Use --dark background with --accent text
2. Light slides: Use --bg background with --text for body
3. Cards: Use --card background with left accent bar in --primary
4. Highlight: Use --primary color for important phrases
