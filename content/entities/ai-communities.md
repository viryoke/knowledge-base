# AI社区与平台索引

## 国际社区

### Reddit r/MachineLearning
- ~280万订阅，最大ML子版块，高质量研究讨论+论文评测

### Hugging Face Community
- 开源AI核心社区，直接与模型创作者交流

### Papers With Code
- [paperswithcode.com](https://paperswithcode.com) — 论文+代码+benchmark排行榜，复现论文必备

### Fast.ai Forums
- [forums.fast.ai](https://forums.fast.ai) — 新手友好，项目式学习文化

## 中文社区

### 知乎 AI话题
- ~500万关注者，中文最大知识分享平台，长文深度回答质量高

### Datawhale
- [github.com/datawhalechina](https://github.com/datawhalechina)
- 多个LLM开源学习项目：llm-action、prompt-guide、leeml-notes
- 中文社区最活跃的LLM学习组织，定期组队学习

### 掘金 AI社区
- [juejin.cn/ai](https://juejin.cn/ai) — 年轻开发者社区，AI应用开发内容活跃

### 极市平台
- [cvmart.net](https://www.cvmart.net) — 国内最专注CV的社区
