## 核心技术点

### StateGraph
- LangGraph 的核心抽象
- 节点（Nodes）：执行具体任务
- 边（Edges）：定义执行流程
- 条件边（Conditional Edges）：基于状态决策

### Agent 模式
- **ReAct Agent**: Reasoning + Acting 循环
- **Tool Use**: 函数调用与外部工具集成
- **Multi-Agent**: 多 Agent 协作系统

### Human-in-the-Loop
- 人类介入审批节点
- 中断与恢复执行
- 状态检查与修改

### RAG Agent
- 结合检索增强生成
- 文档索引与查询
- Agent 驱动的 RAG 流程

## 实战项目

### 基础 Agent
- 简单问答 Agent
- 带工具的 Agent
- 多轮对话 Agent

### RAG Agent
- 文档问答系统
- 检索策略设计
- 答案生成与验证
