# fast.ai Practical Deep Learning for Coders 2022

**讲师**: Jeremy Howard
**平台**: fast.ai
**官网**: [course.fast.ai](https://course.fast.ai)
**YouTube播放列表**: [YouTube](https://www.youtube.com/playlist?list=PLfYUBJiXbdtSvpQjSnJJ_PmDQB_VyT5iU)
**讲数**: 8讲
**总时长**: 约12小时
**难度**: 中级（需要Python基础）
**特点**: 实战导向，自顶向下教学法

## 课程概述

fast.ai 是社区公认的**最佳实战深度学习课程**之一。与传统的"先理论后实践"不同，fast.ai采用**自顶向下**的教学法：

- 第1讲就让你训练出可工作的图像分类器
- 先学会用，再理解原理
- 大量真实项目案例
- 使用 fastai 库（基于 pytorch）

## 为什么选这门课

**适合你的原因**：
1. 你是资深工程师，有足够的编程基础
2. 实战导向符合你的学习风格
3. 12小时可以快速建立深度学习直觉
4. 与李宏毅课程互补（理论 vs 实战）

**社区评价**：
- Reddit r/MachineLearning 常年推荐榜首
- "如果你只能选一门DL课程，选fast.ai"
- 适合有编程经验的人快速入门

## 学习建议

**前置知识**：
- Python基础（你有）
- 基本线性代数概念（矩阵、向量）
- 不需要深入的数学背景

**学习顺序建议**：
1. 先完成 Phase 1 的数学补课（3Blue1Brown线性代数）
2. 然后开始这门课
3. 每讲后动手跑代码（课程提供Jupyter Notebook）
4. 完成课程项目（必须做）

**与李宏毅课程的关系**：
- 李宏毅：理论扎实，理解"为什么"
- fast.ai：实战导向，学会"怎么做"
- 建议两者并行或交替学习
