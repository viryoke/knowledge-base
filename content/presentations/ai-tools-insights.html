PRACTICE 02

让AI工具更懂你

通过个人画像和持久记忆，让AI工具渐进式理解你的偏好。

记忆积累

Hermes Agent 在日常使用中自动记录偏好：字体选择、设计风格、编码习惯、学习进度等

# Hermes 自动记忆示例
user.font.cn: 霞鹜文楷
user.font.code: JetBrains Mono
user.design: Editorial Warm Academic
user.chart: Mermaid only, no ASCII

画像生成

基于记忆生成结构化个人画像，配置在 AGENTS.md 中供所有工具引用

# profile/user-profile.md
role: 12年研发经验 · 架构师
goals: AI/ML · 软考 · 英语
style: 文本优先 · 具体可执行

AGENTS.md 中的引用规则

# 何时引用个人画像
- 生成内容时：参考偏好风格
- 推荐资源时：参考学习目标
- 代码审查时：参考编码习惯

# 何时更新画像
trigger: 用户纠正偏好时
trigger: 发现新偏好信号时
trigger: 定期 lint 检查时

核心理念：记忆是渐进式的。AI工具不会一开始就懂你，但通过持续交互和主动提供个人信息文档，画像会越来越精准。

PRACTICE 04

必装的 Skills

五个经过实战验证的 Skill，覆盖发现、创建、质量控制和设计输出。

find-skills
Skill 搜索引擎。描述你的问题，自动查找是否有现成 Skill 可用。
发现
skill-creator
高质量创建自定义 Skill，自动生成标准目录结构、frontmatter 和文档模板。
创建
superpowers
四步质量流程：头脑风暴对齐思路 → Spec 定义 → Plan 执行 → Verify 验证。
质量
frontend-design
生成 Web 页面时消除"AI味"——避免通用渐变、glassmorphism、过度圆角等典型 AI 生成特征，产出有设计感的界面。
设计 去AI味
ui-ux-pro-max
50+ 风格、161 色彩方案、57 字体配对。搭配 frontend-design 使用：一个设计全局架构，一个生成高质量单页。
素材库
需求描述 → find-skills → 匹配 Skill → superpowers → 高质量输出

PRACTICE 06

富文本的生成

没有订阅图片生成模型？用 Markdown 中转，借助免费工具生成幻灯片和音视频。

工作流 Pipeline

AI 生成 MD → 手动导入 → 富文本输出

AI 工具（Hermes/Claude/GPT）负责内容生成和结构化排版，富文本工具负责视觉呈现。各取所长。

这正是本幻灯片的制作方式 —— 你正在看的这份 slides 就是这套流程的产物。

G

Gemini Canvas

导入 Markdown 内容，利用 Gemini 的多模态能力生成精美幻灯片、信息图和交互式内容

幻灯片 信息图
N

NotebookLM
