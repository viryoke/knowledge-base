## 知乎/掘金热门路线

### 通用AI学习
数学基础(线代/概率/统计) → Python+NumPy → ML基础(西瓜书+统计学习) → DL(d2l+CS231n) → NLP/LLM专项 → 实战项目

### LLM专项
Transformer原理 → Prompt Engineering → RAG → Agent → Finetuning(LoRA等) → 部署(vLLM/ONNX) → 评估
