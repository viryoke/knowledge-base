## 技术英语专项资源

### 代码评审 (Code Review)
**常用表达：**
- "This looks good to me (LGTM)" - 通过
- "Could you clarify the intent here?" - 请求澄清
- "Consider extracting this into a helper function" - 建议重构
- "What's the rationale behind this approach?" - 询问原因
- "Nit: maybe rename this variable to be more descriptive" - 小建议

**学习资源：**
- Google Engineering Practices: [Code Review Guide](https://google.github.io/eng-practices/review/)
- GitHub Blog: [How to write the perfect pull request](https://github.blog/2015-01-21-how-to-write-the-perfect-pull-request/)

### 技术会议 (Tech Meetings)
**常用表达：**
- "Let me walk you through the architecture" - 介绍架构
- "The trade-off here is between X and Y" - 权衡讨论
- "Scalability-wise, this approach..." - 扩展性讨论
- "Let's circle back to this later" - 暂时搁置
- "Action items from this meeting..." - 会议总结

**学习资源：**
- [[english-vocab-tech]] - 技术词汇专题
- TED Talks 技术演讲 - 学习演示结构

### 技术文档 (Technical Writing)
**推荐书籍：**
- *Docs for Developers* by Jared Bhatti et al.
- *Docs Like Code* by Anne Gentle

**在线资源：**
- Google Developer Documentation Style Guide: [style guide](https://developers.google.com/style)
- Microsoft Writing Style Guide: [MS style](https://docs.microsoft.com/en-us/style-guide/welcome/)

## 学习路径建议

### 当前水平评估
基于你的背景（12年技术经验，能读技术文档，听说需提升），建议从**中高级**资源开始。

### 每日学习计划（通勤30-45分钟）

**早上通勤 (15分钟)：**
- 听一集 6 Minute English 或 All Ears English
- 跟读模仿，注意语调

**午休 (10分钟)：**
- 看一个 Rachel's English 发音视频
- 练习1-2个难发音

**晚上通勤 (15-20分钟)：**
- 听 Lex Fridman 或 Software Engineering Daily
- 技术内容，一举两得

**睡前 (10分钟)：**
- Anki 复习词汇（技术英语卡组）
- 读一篇技术博客英文原文

### 每周专项

**周一/三/五：听力**
- 精听一集技术播客，做笔记
- 跟读模仿，录音对比

**周二/四：口语**
- ELSA Speak 练习15分钟
- 或 Cambly/italki 约外教对话

**周六：阅读与词汇**
- 读一篇技术文档或博客
- 整理生词到 Anki

**周日：复习与输出**
- 用英语写技术周记（哪怕只有几句话）
- 复习本周学的表达
